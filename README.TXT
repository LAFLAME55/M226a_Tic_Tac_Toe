TicTacToeULTRAPROMAX

Herzlich Willkommen zu unserem Git Repo für das Spiel TicTacToeULTRAPROMAX :)

Dieses Spiel bzw. Repo ist für unsere LB3 im Modul M226A gedacht. Das ganze Spiel basiert auf Java und ist mittels dem Framework Greenfoot erstellt worden.
Dies war eine Vorgabe der LB des Moduls. 

Kurz zusammengefasst ist unser Spiel ein herkömliches Tic Tac Toe, welches mit kleinen Zusatzfeatures ausgeschmückt wurde. Diese Features werden später genauer
erklärt. 
---------------------------------------------------------------------------------------------------------------------------------------------------------------
Projekt Set up

Hier erklären wir kurz wie man das Spiel aufsetzt. Als erstes muss man Greenfoot installieren (https://www.greenfoot.org/download). 
Nachdem muss dieses Repo klonen (git clone https://github.com/LAFLAME55/M226a_Tic_Tac_Toe.git) und das Projekt in Greenfoot öffnen ("project.greenfoot"
anklicken). Nachdem das Spiel geladen hat, so kann man auf den Button "run" klicken. 
---------------------------------------------------------------------------------------------------------------------------------------------------------------
Infos zum Spiel

Das Spiel läst sich wie das herkömmliche Tic Tac Toe spielen. Dabei muss man 3x den gleichen Charackter in einer Reihe haben (horizontal, vertikal oder diagonal).
Dabei können 2 Spieler spielen. Man muss hierbei mit dem Cursor (derselben Maus) ein Feld anklicken und dann wird der Charackter gesetzt. Sobald dies geschehen ist, kann
man die Maus am 2. Spieler übergeben, sodass er seinen Charackter setzen kann. 

Falls ein Sieger ermittelt wurde, wird ein Winning Screen angezeigt. Gleichzeitig wird sound ausgegeben. Zudem kann man auf den Reset Button drücken, sodass
man nochmals ein Spiel starten kann.
Falls kein Sieger ermittelt wurde, so wird auch ein Bild angezeigt und der Reset Button erscheint auch hier. 

Wenn man das Spiel abrechen möchte, so kann man auf die ESC-Taste drücken und es beginnt ein Sound abzuspielen. Dabei wird versetzt ein Bild angezeigt und
das Spiel wird beendet.
